# Upgrading USB Mini Connectors to USB-C

Old devices using USB-mini connectors were inconvenient as they required carrying separate cables or using adapters.

Removed the original connector and replaced it with a USB-C solder connector that can be used in the same space.

Since the component requires soldering along the PCB edges, it's not easy to solder.

There are more devices like this if I search around, and I plan to upgrade them one by one as time permits.
