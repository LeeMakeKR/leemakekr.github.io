# USB 미니 커넥터를 USB-C 로 변경

usb-mini 커넥터를 쓰는 옛 기기들은 케이블을 따로 챙기거나 젠더를 사용해야 해서 불편했다.

기존 커넥터를 제거하고 커넥터 자리에 쓸 수 있는 usb-c 납땜용 커넥터로 변경.

pcb 모서리를 타고 납땜해야 하는 물건이라 납땜이 쉽지는 않다.

다른 물건들도 찾아보면 더 있는데 시간나는대로 하나씩 변경할 예정
