# 블로그 포스트 템플릿

프로젝트: leemakekr.github.io의 MkDocs 문서 사이트.

## 블로그 포스트 작성 가이드

### 파일 위치
- **한글**: `docs/blog/YYYY-MM-DD-제목.md`
- **영어**: `docs/en/blog/YYYY-MM-DD-title.md`

### 파일명 규칙
- 형식: `YYYY-MM-DD-제목.md` (예: `2026-05-21-led-스트립-게임기.md`)
- 한글 제목은 하이픈으로 연결 (띄어쓰기는 하이픈으로 대체)
- 영문 제목은 소문자 및 하이픈 사용

### 포스트 구조
```markdown
# 포스트 제목

포스트 내용입니다.

## 섹션 1
내용...

## 섹션 2
내용...
```

### 작성 팁
1. **제목**: 명확하고 간결하게
2. **목차**: 필요하면 H2 제목으로 섹션 구분
3. **이미지**: 상대 경로 사용 `![설명](../../assets/img/photo.png)`
4. **코드**: 코드 블록은 언어 식별자 포함: ` ```python`
5. **링크**: 내부 링크는 상대 경로 사용

### 블로그 인덱스 추가
포스트를 작성한 후, 해당 언어의 블로그 인덱스 페이지에 링크를 추가합니다:

**한글 (`docs/blog_index.md`)**:
```markdown
- **[포스트 제목](blog/파일명.md)** - YYYY-MM-DD
  한 문장 요약
```

**영어 (`docs/en/en_blog_index.md`)**:
```markdown
- **[Post Title](blog/filename.md)** - YYYY-MM-DD
  One-line summary
```

### 주의사항
- 포스트는 항상 **최신순(역순)** 으로 정렬
- 한글과 영어 포스트는 동시에 작성해야 함
- 파일명은 소문자 및 하이픈만 사용 (띄어쓰기, 특수문자 금지)
