# Lee Make Log

**Lee Make Log** 는 전자회로, 3D 프린팅, CAD, 메이커 실험들을 기록하는 프로젝트 로그입니다.

---

## 이 사이트는 무엇을 위한 곳인가요?

이곳은 제가:

- 하드웨어 / 전자 프로젝트를 정리하고  
- 3D 프린터와 CAD 실험을 기록하고  
- 펌웨어, 임베디드 시스템, 프로토타입 관련 메모를 정리하고  
- 각종 링크와 트러블슈팅 정보를 모으는  

**개인 작업 노트** 같은 공간입니다.

---

## 최근 작업 영역

- **전자 / 임베디드**
- **3D 프린팅 / CAD**
- **IoT / 홈 오토메이션**

네비게이션 메뉴를 통해 상세 페이지로 이동할 수 있습니다.
